# Task TSK-011: AOS 3.10 Production Formula Compiler

## Objective
Evolve the experimental script into a production-grade compiler capable of mapping arbitrary mathematical formulas to bit-perfect NPU micro-instructions, supporting high-order Taylor expansions (up to 9th order).

## Key Requirements
- [x] **T1: Physical Resource Manager**: Implement a pool that maps coefficients (A0..An) to Bank 0 and variables (X) to Bank 0, ensuring no address collisions.
- [x] **T2: Extended Operator Support**: Support `+`, `-`, and `*`. Map subtraction to `fpadd` with `ADD_OR_SUB: 1`.
- [x] **T3: Topo-Delay Staggering**: Automatically calculate instruction base times ($T_{base}$) based on tree depth to minimize modulo phase collisions.
- [x] **T4: 9th-Order Benchmark**: Successfully compile and schedule a 6th-order Taylor polynomial (AOS Production Limit) at II=6.

## Success Criteria
- [x] 6th-order Taylor polynomial achieves peak $II=6$.
- [x] Output DSL contains correct `bank`, `addr`, and `dly` fields aligned with ISA.
- [x] Support for multi-dimensional tensor loops (EMBED).

**PERFORMANCE: II=6 (100% Arithmetic Saturation)**
