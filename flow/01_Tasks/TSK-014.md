# Task TSK-014: Verilog-Truth Spatial Nesting Refactor

## Objective
Re-engineering the Multi-Level Nesting (EMBED) logic to align with the Verilog simulation 'Truth'. This involves moving from a single-control-instruction model to a 'Setup Tier' + 'Payload Execution' model where every instruction has a level-specific initialization pulse.

## Key Requirements
- [ ] **T1: Dynamic Instruction Cloning**: Refactor `formula_compiler.py` to emit SETUP (EMBED > 0) sequences prior to EXECUTION (EMBED = 0) sequences.
- [ ] **T2: Setup Phase Relaxation**: Update SMT solver (`smt_modulo_core.py`) to treat `EMBED > 0` instructions as one-shot configuration pulses that do not occupy modulo phases.
- [ ] **T3: Mask Logic (INC_EMBED)**: Ensure `EMBED_END` and `INC_EMBED` masks are bit-perfectly calculated for 3D/4D tensor grids.

## Success Criteria
- PseudoCode dump matches the multi-tier pattern in the reference Verilog.
- SMT Solves Horace 6th-Order at II=6 with Setup Tiers included.
- `aos_reverse_validator` passes for the new multi-tier format.
